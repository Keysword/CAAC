# CAAC 备考库

> 中国民用航空局（CAAC）相关考试备考资料与知识整理

## 📁 目录结构

| 文件 | 说明 |
|------|------|
| [README.md](./README.md) | 本文件，备考库总览与使用指南 |
| [错题本.md](./错题本.md) | 刷题过程中遇到的错题记录与解析 |
| [知识文档.md](./知识文档.md) | 规范、知识点、易混淆概念的归纳总结 |

## 🎯 使用方式

1. **刷题时** → 遇到错题或不确定的题目，记录到 `错题本.md`
2. **遇到新知识点/规范** → 整理到 `知识文档.md`
3. **定期复习** → 回顾错题本和知识文档，强化记忆
