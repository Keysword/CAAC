### 题号/来源：力矩方向 — 机翼升力对重心的力矩
- **题型**：单选
- **错误日期**：2025-06-16
- **题目内容**：
  > 常规布局的飞机，机翼升力对飞机重心的力矩常为使飞机机头的（）力矩？
  > A. 上仰  B. 下俯  C. 偏转
- **我的答案**：A（上仰）
- **正确答案**：B（下俯）
- **错误原因**：
  - [x] 概念不清：混淆了升力方向与力矩方向
- **知识点定位**：见 [知识文档.md](./知识文档.md) #飞行原理与流体力学基础
- **解析**：
  > 常规布局中，重心 CG 被设计在气动中心 AC **之前**（即 AC 在 CG 之后）。升力作用点 AC 在重心 CG 之后，升力向上 → 力矩使后端上抬、前端下沉 → **机头下俯**。
  >
  > 注意：题目解析本身有术语错误（"上反角"应为"安装角"），但答案（下俯）是正确的。不能用"升力向上所以机头上仰"的直觉判断，必须看力矩方向。
- **举一反三**：
  - 水平尾翼的向下力产生的是**上仰**力矩（与机翼下俯力矩平衡）
  - 失速改出时推杆减小迎角，也是利用下俯力矩原理
  - 纵向静稳定性要求 CG 在 AC 之前，由此产生的下俯力矩是飞机"自动恢复低头"的物理基础

### 题号/来源：下滑飞机升力与重力关系
- **题型**：单选
- **错误日期**：2025-06-16
- **题目内容**：
  > 对于下滑中的飞机来说，升力和重力关系是（）
  > A. L = Wgcosα  B. L = Wgsinα  C. L = Wg
- **我的答案**：B（L = Wgsinα）
- **正确答案**：A（L = Wgcosα）
- **错误原因**：
  - [x] 概念不清：混淆了 sin/cos 的分解方向
  - [x] 审题失误：把下滑角当成了迎角
- **知识点定位**：见 [知识文档.md](./知识文档.md) #飞行原理与流体力学基础 + 受力分析图 [gliding_forces.png](./gliding_forces.png)
- **解析**：
  > 稳定下滑中，飞机匀速运动，力沿**飞行轨迹方向**分解平衡：
  > - **垂直于轨迹**：升力 L = Wg·cos(γ)（重力垂直于轨迹的分量）
  > - **沿轨迹方向**：阻力 D = Wg·sin(γ)（重力沿轨迹的分量）
  >
  > γ 是**下滑角**（飞行轨迹与水平面的夹角），不是迎角。题目解析写"α是迎角"是术语错误。下滑时升力小于重力（cosγ < 1），因为重力有一部分分力"帮"飞机往下滑。
- **举一反三**：
  - 平飞时：L = Wg（轨迹水平，重力全垂直于轨迹）
  - 爬升时：L = Wg·cos(γ)（公式形式同下滑，γ为爬升角）
  - 记忆：垂直分量用cos，平行分量用sin；升力垂直于轨迹，所以升力=cos
